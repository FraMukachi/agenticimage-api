# AgenticImage API

AI-powered image processing with specialized agents: Brain (classification), Vision (detection), Heart (restoration), and Hands (upscaling).

## 🚀 Quick Start

### Web Interface
Visit [agenticimage.vercel.app](https://agenticimage.vercel.app)

### JavaScript SDK
```html
<script src="https://agenticimage.vercel.app/sdk.js"></script>
<script>
  AgenticImage.process(file).then(result => {
    console.log(result.upscaled);
  });
</script>
